🔧 Troubleshooting

  • Backend not responding? Make sure you ran setup-backend-simple.bat
  • Frontend not loading? Make sure you ran setup-frontend.bat
  • Admin panel 404? Check if backend server is running on port 8000
  • CORS errors? Make sure both servers are running on correct ports
  • Database errors? Delete backend/db.sqlite3 and re-run migrations
