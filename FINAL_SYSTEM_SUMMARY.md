## 🔧 **QUICK START**

### **Option 1: One-Click Launch**
```bash
# Start both backend and frontend
START_EVERYTHING.bat
```

### **Option 2: Manual Launch**
```bash
# Backend (Terminal 1)
cd backend
python manage.py runserver

# Frontend (Terminal 2)  
cd frontend
npm run dev
```

### **Option 3: Train AI Model**
```bash
# Interactive AI training
TRAIN_AI_MODEL.bat
```
